# EDF Pricing

Currently just a draft, but should be a set of script to help you choose which contract from EDF works best for you
depending on your previous data consumption.

## Getting started

Depending on your os, run the `create_uvenv` (linux) or the `create_uvenv.ps1` (windows) to install/update uv and setup
a python version and a working python environment. By default, environments are stored at `$HOME/ENV`.
